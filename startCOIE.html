Chroniques Oubliées Fantasy : script import/export

Description

À l'origine de cette initiative, ce script est une modification de celui réalisé par Kamille. Les fiches, quant à elles, ont été faites à partir des fonctionnalités du scipt d'Ulti. Je vous invite aussi à consulter le site co-drs pour qui offre un grand nombre d'options pour la gestion de vos parties et l'import sur Roll20.

Pré-requis

Ce document a pour but de vous aider à installer et configurer le script COIE sur le site Roll20.net.

Liste des pré-requis:

  • Un compte Pro sur Roll20 afin de pouvoir intégrer des scripts
  • Un niveau de connaissance minimum sur Roll20.
  • Une partie configurée avec la "Feuille de Personnages : Chroniques Oubliées Fantasy (French)" par Nata.
  • Une partie configurée avec le script COFantasy par Ulti.

C'est parti !
