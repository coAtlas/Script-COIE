Importer des fiches de personnages

Importez tous les personnages dont vous avez besoin pour vos parties de Chroniques Oubliés en seulement 5 étapes (voir plus bas) !

Informations :

  • Nota Bene : vous trouverez dans la partie "Capacités" de la fiche des informations complémentaires ou des limitations aux capacités
  • Bestiaire : si vous importez des créatures du bestiaire, il est conseillé de prendre connaissance de ce qui est décrit dans le livre. Certaines informations ne figure pas sur les fiches
  • COFantasy : il est important de consulter le script pour comprendre le fonctionnement des capacités automatisés et leurs limites dans certains cas.
  • Détail : positionnez votre souris sur le nom du personnage pour le voir en entier si ce n'est pas le cas. Positionnez votre souris sur l'image du personnage pour voir sa description
  • Me contacter : si vous constatez des problèmes avec la fiche, n'hésitez pas à me faire remonter l'information via le forum.

Étapes :

  1. Créer l'handout : sur Roll20, dans votre partie, créez un handout nommé "handoutCOIE".
  2. Sélectionner les fiches à importer : sélectionnez les personnages que vous voulez importer en cliquant sur leur nom. Ce dernier deviendra alors rouge. Si vous souhaitez retirer un personnage, rien de plus facile : cliquer à nouveau sur leur nom qui redeviendra vert.
  3. Copier : une fois que vous avez terminé votre sélection, cliquez sur le bouton "Copier" ci-dessous ou alors cliquez dans la zone de texte et faites : Ctrl + A puis Ctrl + C.
  4. Coller : retournez sur Roll20, faites modifier l'handout "handoutCOIE", puis collez ce que vous avez copié.
  5. Importer : tapez dans le chat la commande "!coie-import".

Version de la base de données COIE : 1.3
